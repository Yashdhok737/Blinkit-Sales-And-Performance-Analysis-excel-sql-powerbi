# Blinkit Sales & Performance Analysis
### Project Overview

This project focuses on analyzing Blinkit's sales, customer behavior, and outlet performance using Excel, SQL, and Power BI.


###  Objectives
- Analyze Blinkit sales performance across different outlet types and locations.
- Identify top-performing product categories and customer preferences.
- Compare sales based on fat content, outlet size, and establishment year.
- Clean and preprocess raw data using Excel for accurate analysis.
- Perform business queries and insights extraction using SQL.
- Create interactive dashboards in Power BI for data visualization.
- Generate actionable insights to support business decision-making.


###  Tools & Technologies Used
- Microsoft Excel → Data cleaning and preprocessing
- SQL (PostgreSQL) → Data querying and transformation
- Power BI → Data visualization and dashboard creation


### Dataset Information

- The dataset contains information related to:

  - Item details
  - Outlet information
  - Item type and category
  - Sales performance
  - Customer ratings

  - Fat content

### Key Business Insights

- Tier 3 outlets generated higher sales compared to other locations.
- Regular fat products contributed more revenue than low-fat products.
- Supermarket Type 1 outlets showed the highest overall sales performance.
- Certain item categories consistently performed better across all outlet sizes.
- Older outlets demonstrated stronger customer trust and stable revenue generation.
